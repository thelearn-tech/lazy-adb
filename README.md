# lazy-adb
ADB for lazy People

# Installation

***git clone https://github.com/thelearn-tech/lazy-adb***

***cd lazy-adb***

***chmod +x lazy-adb.sh***

***sh lazy-adb.sh***

If **sh** didn't work  try **./** or **bash**

# Requirements

**Bash**

**ADB**
